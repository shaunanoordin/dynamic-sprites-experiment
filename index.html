Dynamic Sprites Experiment

  • In the image above, the <canvas> has HTML width=160 and height=80. However, it's scaled up since it has CSS width=100%. The sprite art still retains pixel sharpness, because we used canvas { image-rendering: pixelated } (CSS)
  • The Marios are rendered using canvasContext2d.drawImage(). The first (left-most) is drawn at 1.0x scale. The second is drawn at 4.0x scale. The second image still retains pixel sharpness, because we used canvasContext2d.imageSmoothing = false (JS)
  • Tested on macOS + Chrome 102 / Firefox 101.
